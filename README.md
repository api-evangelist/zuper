# Zuper

Zuper is an intelligent field service management platform that consolidates job scheduling, dispatching, customer management, invoicing, timesheets, and field team operations into a single AI-powered operating system. The Zuper REST API gives developers programmatic access to all platform data across jobs, assets, projects, inventory, contracts, and financials.

## Authentication

All API requests require an API key in the `x-api-key` header. Generate keys from Settings > Developer Hub > API Keys in the Zuper dashboard.
